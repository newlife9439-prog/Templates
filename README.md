# Third Template

This project contains a small HTML/CSS template.

## Quick Start

1. Install dev dependencies:

   npm install

2. Format files with Prettier:

   npm run format

3. Lint HTML and CSS:

   npm run lint:html
   npm run lint:css

4. Run both checks:

   npm run check

## Notes
- Config files added: `.editorconfig`, `.prettierrc`, `.htmlhintrc`, `.stylelintrc.json`.
- You can open the `index.html` in a browser for local preview.
